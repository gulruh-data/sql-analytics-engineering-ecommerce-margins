# 💰 E-commerce Financial Analytics Engineering (SQL)

## 📌 Project Purpose
Bu proje, bir e-ticaret sitesinin (Greenweez) ham satış verilerini, gelişmiş finansal metriklere dönüştürmek için tasarlanmıştır. Temel odak noktası, iş birimleri için **standartlaştırılmış hesaplama yöntemleri** oluşturmaktır.

## 🚀 Specialist-Level Skills Demonstrated
- **Business Logic Automation:** Kâr marjı ve promosyon derinliği hesaplamaları SQL seviyesinde standardize edildi.
- **Edge Case Handling:** `SAFE_DIVIDE` mantığı ile veri hatalarının (sıfıra bölünme vb.) önüne geçildi.
- **Data Enrichment:** Ham satış verileri, ticari karar vermeyi destekleyecek yeni KPI'lar ile zenginleştirildi.

## 📊 Business Metrics Defined
1. **Margin %:** Ürün bazlı kârlılık oranı.
2. **Promo %:** Uygulanan indirimin liste fiyatı üzerindeki etkisi.
